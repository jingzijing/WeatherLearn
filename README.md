# WeatherLearn
android平台 天气预报APP

### 说明
##### APP使用的天气数据源是彩云天气的API，需要你注册自己的token，这是api所需要的

##### 彩云天气开放平台地址：
https://dashboard.caiyunapp.com/user/sign_in/

##### 获取到token后替换到下图位置中即可运行项目
